npm

NPM是随同NodeJS一起安装的包管理工具，能解决NodeJS代码部署上的很多问题，常见的使用场景有以下几种：
允许用户从NPM服务器下载别人编写的第三方包到本地使用。
允许用户从NPM服务器下载并安装别人编写的命令行程序到本地使用。
允许用户将自己编写的包或命令行程序上传到NPM服务器供别人使用。

主要的命令：
npm install:安装指定的包，后面直接跟包名（后面不跟任何参数表示根据配置文件 package.json安装所有包）。安装分为全局安装和本地安装，区别为全局安装需要在最后跟上'-g'，全局安装的能在所有目录下使用，而本地安装的只 能在安装目录下运行安装的包或命令。安装指定版本为：npm install grunt-cli@0.1.9。
npm uninstall:卸载指定的包，卸载指定版本的同上。
npm ls:查看当前目录安装了哪些包
